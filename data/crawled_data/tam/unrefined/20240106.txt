Tamalpais CrossFit WOD January 6 2024
 


GRIZZLY
– Teams of 2- 3 sets (each/1:1)

200m Run

8 Power Cleans 95/135 M75/105

200m Run

8 Power Cleans 95/135 M75/105

200m Run


KODIAK
– Teams of 2- 3 sets (each/1:1)

200m Run

5 Power Cleans 95/135 M75/105

200m Run

5 Power Cleans 95/135 M75/105

200m Run


POLAR
– 3 sets

150m Run

8 Power Cleans (light)

150m Run

8 Power Cleans (light)

150m Run

rest 1:1 between sets


ABS- All Levels:

3 Rounds (40 seconds on/20 seconds)

Plank Pull Through [use a weight you can control and brace core recommended 35/25 Dumbbell]

-Rest 1:00-

3 Rounds (40 seconds on/20 seconds)

Max Wall Ball Throw Sit Ups

*Use a weight that allows for a non-stop effort
